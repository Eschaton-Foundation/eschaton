# Naming Conventions

| Type                 | Format/Prefix                | Example(s)                              |
|----------------------|------------------------------|-----------------------------------------|
| PHP Functions        | `cwp_[module]_[action]()`    | `cwp_universal_toggle_setting_callback()`|
| JavaScript Functions | `cwp[Module][Action]()`      | `cwp_show_floating_notification()`      |
| Option Keys          | `cwp_[feature]_settings`     | `cwp_authnet_api_keys`, `cwp_fm_cache_data` |
| Nonce Actions        | `cwp_[feature]_nonce`        | `cwp_universal_nonce`                   |

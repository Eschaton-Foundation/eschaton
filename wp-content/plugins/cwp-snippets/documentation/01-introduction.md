
# Introduction

Welcome to CWP Snippets! This document will provide you with an overview of the plugin and its features.

## What is CWP Snippets?

CWP Snippets is a powerful plugin that allows you to extend the functionality of your WordPress site by adding custom code snippets. These snippets can be used to display data from a FileMaker database, create custom forms, add custom styling, and much more.

## Core Features

*   **Snippet Management:** Easily create, edit, and manage your code snippets from the WordPress admin area.
*   **Snippet Types:** CWP Snippets supports four main types of snippets: Snippets (which include Templates and Samples), Functions ( Pro ), Styles ( Pro ), and Scripts ( Pro ).
*   **Shortcode Support:** Use shortcodes to display your snippets anywhere on your site.
*   **FileMaker Integration:** Seamlessly connect to your FileMaker database to display data on your WordPress site.
*   **Import/Export ( Pro ):** Easily import and export your snippets.

## Who is this for?

CWP Snippets is for WordPress developers, designers, and site builders who want to extend the functionality of their sites without having to edit theme files directly. It is also for FileMaker developers who want to integrate their FileMaker solutions with WordPress.
